## Built With

- React JS
- React Bootstrap
- SaSS

<br> </br>


### Installing

To install, type the following command: 

```bash
npm install --force
```

Currently facing some issues with the npm packaging. 

<br> </br>


### Running

To run the app, type the following command:

```bash
npm run start
```
